# オセロ棋譜検討

ブラウザだけで動く、オセロの対局・CPU対戦・棋譜検討アプリです。  
対局後に1手ずつ振り返り、AI評価、候補手、悪手一覧、解説、対局全体の総評を確認できます。

GitHub Pagesでそのまま動作します。サーバーやDBは使っていません。

## Features

- 2人対戦 / CPU対戦
- CPU難易度 easy / normal / hard
- 棋譜の1手再生
- AI推奨手との比較
- 「次の分岐へ」でミスした手へジャンプ
- 候補手ランキング
- 悪手一覧
- 1手ごとの解説
- 対局全体の総評
- JSON形式の棋譜読み込み・書き出し
- PC / スマホ対応UI

## How to Use

1. `index.html` をブラウザで開く
2. 2人対戦またはCPU対戦でオセロを打つ
3. 対局後に「検討する」を押す
4. 1手ずつ、AI評価と解説を見ながら振り返る

ローカルサーバーで確認する場合:

```sh
python3 -m http.server 8000
```

```text
http://127.0.0.1:8000/index.html
```

## Notes

AI評価と解説はすべてブラウザ内で生成しています。  
強豪AIではなく、角・危険マス・合法手数・終盤石数などを使った学習用のローカル評価AIです。
